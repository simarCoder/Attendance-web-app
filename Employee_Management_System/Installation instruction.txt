EMPLOYEE MANAGEMENT SYSTEM
Version 1.0

Thank you for using this software.

This application is designed to manage:
- Employee records
- Attendance tracking
- Salary calculation
- Employee document storage

--------------------------------------
SYSTEM REQUIREMENTS
--------------------------------------
- Windows 10 or Windows 11
- Minimum 4GB RAM
- 200MB free disk space

--------------------------------------
INSTALLATION INSTRUCTIONS
--------------------------------------
1. Download the ZIP archive.
2. Right-click the ZIP file and select "Extract All".
3. Open the extracted folder.
4. Double-click the file: Employee_Management_System.exe
5. If Windows shows a security message:
   Click "More info" and then "Run anyway".

--------------------------------------
FIRST TIME SETUP
--------------------------------------
- The application will automatically create its local database.
- Do not delete the database file from the application folder.
- Keep all files inside the same folder.

--------------------------------------
IMPORTANT NOTES
--------------------------------------
- This software runs locally on your system.
- No internet connection is required.
- All employee data is stored on your computer.
- Please do not rename or move internal application files.
